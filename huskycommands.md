# Referencia de Commits para Husky

| **Tipo**     | **Cuándo usarlo**                                | **Ejemplo para tu App**                       |
| :----------- | :----------------------------------------------- | :-------------------------------------------- |
| **feat**     | Una nueva funcionalidad o componente.            | `feat: implement product search logic`        |
| **style**    | Cambios de CSS, colores, fuentes o responsive.   | `style: mobile-first layout for home screen`  |
| **fix**      | Arreglar un bug o un error de TypeScript.        | `fix: resolve hydration error in scanner`     |
| **chore**    | Configuración (Husky, librerías, tsconfig).      | `chore: update path aliases in tsconfig`      |
| **refactor** | Mejorar código existente sin cambiar su función. | `refactor: simplify barcode validation logic` |
| **docs**     | Cambios en el README.                            | `docs: add installation steps to readme`      |
